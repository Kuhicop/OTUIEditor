## About

Visual OTUI designer with .OTUI file generator.
[YouTube Video](https://www.youtube.com/watch?v=CQBn6jFqhlI)

## Build

- [Download and install Qt 5.13.0+](https://www.qt.io/download-qt-installer)
- Open Qt Creator
- Import project file
- Build
